# Tuski
![Imgur](https://i.imgur.com/UfuFOTf.jpg)

Tsuki is a **node.js** based website. Made for web novel, light novel, and novels hosting were you can store your novels.

# Setup

```
git clone https://github.com/herakumi/tsuki

cd tsuki

npm i

```
